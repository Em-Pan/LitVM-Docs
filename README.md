# README

Welcome to the official documentation for **LitVM** — the first trustless EVM-compatible rollup secured by Litecoin.

## What is LitVM?

LitVM brings smart contract capabilities to Litecoin through a zero-knowledge rollup architecture. It combines:

* **Arbitrum Orbit** for EVM compatibility
* **BitcoinOS Grail Bridge** for trustless LTC bridging
* **Litecoin's security** as the foundation
