## Block Explorer

The LitVM Testnet Block Explorer is powered by **Caldera** and provides full visibility into the network.

**URL:** [https://liteforge.explorer.caldera.xyz](https://liteforge.explorer.caldera.xyz)

### Features

* **View Transactions** — Track transaction status, gas used, and confirmation times
* **Explore Blocks** — See block details including timestamp, transactions, and gas limit
* **Account Lookup** — Check balances and transaction history for any address
* **Contract Verification** — Verify and interact with deployed smart contracts
* **Event Logs** — View emitted events and logs from contract interactions

### How to Use

1. Visit [https://liteforge.explorer.caldera.xyz](https://liteforge.explorer.caldera.xyz)
2. Enter a transaction hash, block number, or wallet address in the search bar
3. View detailed information about the transaction, block, or account
