# Indexing LitVM with Goldsky

Goldsky handles the hard parts of building on crypto rails: real-time data and reliable connectivity, so you can ship better products, faster.

## Overview

Subgraphs give you instant, queryable APIs for blockchain data. Define the events and entities you care about, and Goldsky handles the indexing, hosting, and maintenance, delivering a production-ready GraphQL endpoint with zero infrastructure overhead.

Turbo is Goldsky's next-generation pipeline engine — built for teams that need maximum performance. It delivers really fast decoding, infinite filters, dynamic tables, and live data inspection, so you can move from raw chain data to production insights without the wait. Instead of querying an API, you get the data directly in your database, shaped the way you need it, updating continuously as new blocks are produced.

## Goldsky Products

### Subgraphs

Instant GraphQL APIs for blockchain data with zero maintenance.

* **Fast Queries:** Optimized for low latency and high throughput.
* **Customizable:** Tailor indexing logic to your app's exact needs.
* **DX:** Webhooks, tagging, and versioning keep data clean as your project evolves.

**Typical Use Cases:** dApps, NFT marketplaces, gaming, DAOs, or any app needing reliable onchain data.

### Turbo Pipelines

Next-generation pipeline performance for high-throughput chains.

* **Really Fast Decoding:** Purpose-built for chains where speed isn't optional.
* **Infinite Filters:** Granular control over exactly what data you capture and how.
* **Dynamic Tables & Live Inspection:** Structure data on the fly and observe pipelines in real time.
* **AI Skills:** Leverage AI to deploy, debug, and query your pipelines.

**Typical Use Cases:** High-frequency trading data, real-time leaderboards, performant data stacks on fast chains, advanced analytics, or any use case where apps require rich onchain data joined with other datasets.
